# ✈️ Airline Reservation System (C++)

## 📌 Overview
This project is a **C++ based Airline Reservation System** that allows passengers to book and manage flight tickets, while providing administrators with options to manage flights and view ticket details.  
It is a **console-based application** that uses **file handling** for persistent storage.

---

## 🚀 Features

### 👤 Passenger Module
- Register a new passenger  
- Login with username & password  
- View personal profile  
- View available flights  
- Book tickets (with unique Ticket ID)  
- View all bookings  
- View detailed ticket information  
- Cancel bookings  

### 🛫 Admin Module
- Login with default admin credentials  
- View admin profile  
- Add new flights  
- View all flights  
- View all issued tickets  

---

## 🗂️ File Storage
The system stores data using text files:
- **flights.txt** → Stores flight details  
- **passengers.txt** → Stores passenger details and bookings  
- **tickets.txt** → Stores ticket details  

This ensures data persistence across multiple runs of the program.
